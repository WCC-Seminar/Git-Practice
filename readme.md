### local rules?

* 初めてある問題を解く人は:
 + その問題の問題文を適切な場所に写し(前から解いていく必要はないから飛んでもよろしい),
 + 例えば「リストは a = [a1, a2,.. an] で与えられるとして」みたいな，みんなで揃えると幸せになりそうなのもあれば追記し，
 + 自分の解答を記入する．
